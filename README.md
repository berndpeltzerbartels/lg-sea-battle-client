# LG Sea Battle

Small Babylon.js prototype for a 3D boat scene on open water.

## Run

```sh
npm install
npm run dev
```

Open `http://localhost:5173/`.

## Controls

- Up arrow: accelerate
- Down arrow: reverse / slow down
- Left arrow: turn left
- Right arrow: turn right

The boat uses smoothed acceleration, drag, turn velocity, camera follow, and a small wave bob so movement feels gradual instead of jerky.
